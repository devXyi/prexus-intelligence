# Prexus Core | Sovereign Intelligence Gateway

Prexus Core is a high-performance, Zero-Trust backend written in Go. It serves as the computational kernel for the Prexus Sovereign Intelligence Platform.

## Architecture Philosophy
- **Compute:** 10,000-iteration stochastic Monte Carlo simulations (VaR / CVaR).
- **Security:** Attribute-Based Access Control (ABAC) with Argon2id & short-lived JWTs.
- **Immutability:** SHA-256 cryptographically hash-chained append-only audit ledger.
- **Deployment:** Cloud-native, stateless, and Docker-ready for AWS GovCloud / Azure Gov.

## Directory Structure
- `/cmd/api-gateway`: TLS termination, API routing, and HTTP handlers.
- `/internal/kernel`: The proprietary mathematical models (Meteorium).
- `/internal/audit`: The tamper-evident cryptographic ledger.
- `/internal/middleware`: Zero-Trust policy enforcement.
